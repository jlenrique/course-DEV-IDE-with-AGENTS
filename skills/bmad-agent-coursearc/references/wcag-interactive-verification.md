# WCAG 2.1 AA Verification for Interactive Content

Check every interactive lesson for:

- Keyboard-only navigation through all controls.
- Visible focus indicators and logical tab order.
- Contrast ratios meeting AA thresholds.
- Text alternatives for non-text content.
- Clear instructions that do not rely on color alone.
- Captions/transcripts for audio or video elements.

Report format:
- criterion
- pass/fail
- evidence
- remediation action
