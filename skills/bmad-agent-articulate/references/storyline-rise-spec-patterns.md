# Storyline and Rise Spec Patterns

For every module:

1. Define page/scene map and learning objective coverage.
2. Specify interaction type per section (timeline, tabs, click-to-reveal, scenario).
3. Define completion criteria and navigation constraints.
4. Map feedback behavior for correct/incorrect pathways.
5. Specify accessibility requirements up front (keyboard path, focus order, text alternatives).

Output format:

- module map
- interaction matrix
- media inventory
- publish settings recommendation
