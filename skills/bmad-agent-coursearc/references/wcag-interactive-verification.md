# WCAG 2.1 AA Verification for Interactive Content

Reference documentation:

- WCAG 2.1 Quick Reference: https://www.w3.org/WAI/WCAG21/quickref/

Check every interactive lesson for:

- 2.1.1 Keyboard (A): keyboard-only navigation through all controls.
- 2.4.7 Focus Visible (AA): visible focus indicators and logical tab order.
- 1.4.3 Contrast (Minimum) (AA): contrast ratios meeting AA thresholds.
- 1.1.1 Non-text Content (A): text alternatives for non-text content.
- 1.3.3 Sensory Characteristics (A): instructions do not rely on color alone.
- 1.2.2 Captions (Prerecorded) (A): captions/transcripts for audio or video elements.

Report format:
- criterion
- pass/fail
- evidence
- remediation action

Completion rule:

- Do not mark accessibility verification complete until every criterion has evidence or an active remediation entry.

CourseArc interaction note:

- Validate each supported interaction type (sorting, flip card, virtual patient) against all criteria above and record evidence per interaction type.
