# Canva Capability Catalog

This catalog maps Canva capabilities to educational production use cases.

## What Canva Can Do Reliably in This Repo Context

1. Import PPTX assets for manual enhancement.
2. Apply brand templates and visual styling in the editor.
3. Produce static and motion exports (PNG, PDF, MP4) through UI workflows.
4. Support accessibility-aware design checks (contrast, font scale, layout clarity) via guided review.

## What Canva Cannot Do Reliably Here

1. Programmatic element-level editing in this repository runtime.
2. Deterministic API-driven visual composition for full production automation.
3. Woodshed-style automated reproduction scoring.

## Use Case Matrix

| Use Case | Canva Contribution | Output |
|---|---|---|
| Infographic for module concept | Template selection + manual editor build | PNG/PDF export |
| Slide visual enhancement after Gamma | PPTX import + layout cleanup + iconography | PPTX/PNG export |
| Social summary card for course updates | Brand template adaptation | PNG export |
| Motion opener card | Animated elements in editor | MP4 export |
