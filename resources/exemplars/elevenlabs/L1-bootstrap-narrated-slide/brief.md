# L1 Bootstrap Narrated Slide

## Purpose

Bootstrap Story `3.4` with a repo-owned ElevenLabs exemplar that proves the minimum narration path:

- script in
- MP3 out
- VTT out
- timing metadata available for downstream pipeline use

## Source Material

- Primary script source: `course-content/staging/story-3.2-samples/NS-C1M1L1-02-narration-script.md`
- Intended use: a small, single-segment or single-slide narration smoke path

## What This Exemplar Proves

- style-guide voice defaults can drive narration without ad hoc parameter guessing
- timestamped narration output can be converted into a usable VTT file
- the Voice Director / `elevenlabs-audio` path can produce files Marcus can route forward

## Notes

This is a bootstrap exemplar created from repo-owned material so Story `3.4` can proceed without blocking on a new human-supplied exemplar. A future Juan-provided exemplar should supersede or complement it.
