# Run Records

This folder contains run records for trial runs of the course-DEV-IDE-with-AGENTS project code. These records serve as documentation of trial runs, including user notes and observations from the BMAD party mode consultation sessions.

## File Naming Convention

All run record files must be timestamped using ISO format in the filename:

```
run-record-YYYY-MM-DDTHH-MM-SS.md
```

Example: `run-record-2026-04-15T20-44-30.md`

## Content Structure

Each run record should include:
- Timestamp metadata at the top of the file
- Description of the trial run
- User notes and observations
- BMAD party mode agent recommendations and decisions
- Any issues encountered and resolutions

## Template

```markdown
---
timestamp: 2026-04-15T20:44:30
run_type: trial
participants: [user, bmad-agents]
---

# Trial Run Record - [Brief Description]

## Context
[Description of what was being tested]

## User Notes
[User observations and notes]

## Agent Recommendations
[BMAD party mode agent input]

## Decisions Made
[Key decisions from the session]

## Next Steps
[Follow-up actions]
