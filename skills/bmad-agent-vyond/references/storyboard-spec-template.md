# Vyond Storyboard Spec Template

Use this structure for every Vyond request:

1. Lesson context: course, module, lesson, learning objective IDs.
2. Runtime target: total seconds and per-scene duration.
3. Scene list:
- scene_id
- instructional purpose
- narration segment reference
- on-screen visual objective
- required on-screen text (if any)
4. Character/cast profile: role, attire, expression, motion limits.
5. Camera/motion notes: cuts, pans, transitions.
6. Accessibility notes: contrast, text size, caption-safe layout.
7. Export target: 1080p MP4 plus source project naming convention.
