# Vyond Studio Build Playbook

1. Confirm storyboard and narration timing are approved.
2. Create project using approved aspect ratio (default 16:9).
3. Build scenes in storyboard order.
4. Apply style-bible visual constraints before adding decorative elements.
5. Add text only where storyboard requires it; avoid dense text slides.
6. Align animation timing to narration beats.
7. Run final pass for readability, pacing, and visual continuity.
8. Export MP4 and record artifact path for Marcus handoff.
