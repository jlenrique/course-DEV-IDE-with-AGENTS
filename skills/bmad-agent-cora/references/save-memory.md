# Save Memory

Immediately persist the current dev-session context to the memory sidecar at `{project-root}/_bmad/memory/cora-sidecar/`.

## Process

Update `index.md` with:

- Current session anchor commit and branch
- Outstanding pre-closure findings (if any)
- Intended next anchor task for the next session
- Any operator preferences expressed this session

If a cross-session pattern has crystallized (3+ confirming observations), append to `patterns.md`.

Append a one-line entry to `chronology.md` with `YYYY-MM-DD HH:MM — <summary>`.

## Output

"Memory saved. [brief summary of what was updated]"
