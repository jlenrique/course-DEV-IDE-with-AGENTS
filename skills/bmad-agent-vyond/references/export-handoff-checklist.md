# Vyond Export and Handoff Checklist

- Export format: MP4, 1080p.
- Filename includes course/module/lesson and revision number.
- Export duration matches approved storyboard runtime.
- Captions/transcript handoff note included if narration is external.
- Handoff payload includes:
  - output file path
  - duration seconds
  - scene coverage summary
  - known limitations/issues
  - recommended next specialist
