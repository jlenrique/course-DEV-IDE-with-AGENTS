# Midjourney Discord/Web Iteration Workflow

1. Start with baseline prompt and fixed seed.
2. Generate first grid; evaluate against learning objective intent.
3. Use variation/upscale paths intentionally:
- V for concept divergence.
- U for final candidate refinement.
4. Record each iteration with prompt, seed, and rationale.
5. Stop when acceptance checks pass; do not continue novelty loops.
6. Return final prompt package to Marcus:
- final prompt text
- parameter set
- seed
- selected image links
- usage notes
