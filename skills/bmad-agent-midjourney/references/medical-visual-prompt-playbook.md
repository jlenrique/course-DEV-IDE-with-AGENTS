# Medical Visualization Prompt Playbook

Prompt blocks:

1. Subject block: anatomy/process/clinical context.
2. Composition block: layout, perspective, focal hierarchy.
3. Visual standards block: style-bible constraints (palette, tone, typography exclusion).
4. Safety block: realism constraints and forbidden depictions.
5. Parameter block: v6/v7 controls.

Quality checks:

- Accuracy: no impossible anatomy or contradictory labels.
- Legibility: key concept is clear within 3 seconds.
- Professional tone: avoid sensational imagery for instructional contexts.
- Reusability: prompt version and seed are logged for repeatability.
