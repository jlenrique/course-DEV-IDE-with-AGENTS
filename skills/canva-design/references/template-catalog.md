# Canva Template Catalog (Educational)

Use this catalog to pick template families aligned with instructional intent.

## Selection Heuristics

1. Prioritize instructional clarity over decorative complexity.
2. Match color and typography to resources/style-bible/ standards.
3. Keep heading hierarchy explicit and scannable.
4. Reserve high-motion templates for short emphasis segments.

## Recommended Template Families

| Content Need | Recommended Template Traits | Avoid |
|---|---|---|
| Learning objective explainer | clean title, two-column body, icon slots | dense decorative backgrounds |
| Process flow infographic | numbered path, clear connector spacing | ambiguous directional lines |
| Clinical case summary | card stack with consistent labels | mixed font families and low contrast |
| Assessment reminder card | high-contrast callout and concise bullets | paragraph-heavy blocks |

## Template Search Workflow

1. Open Canva templates and start with these search terms:
  - "education infographic"
  - "medical presentation"
  - "clinical case study"
  - "learning objectives"
2. Filter for layout-first templates (clear title area, structured body regions, predictable card spacing).
3. Keep 2-3 finalists and evaluate against the review checklist below.
4. Record selected template name in the design handoff so future iterations can reuse it.

## Review Checklist

1. Contrast meets accessibility standards.
2. Font sizes remain readable in projected and mobile contexts.
3. Visual hierarchy matches instructional priority.
4. Export format chosen for downstream platform needs.
