# L1 Course Structure Snapshot

## What This Exemplar Is

A baseline Canvas course-structure snapshot used to verify that the Canvas specialist can authenticate, query course metadata, and capture module/page/assignment/quiz inventories through the Canvas API.

## Why It Matters

This is the minimum viable woodshed benchmark for Story 3.6:
- proves live Canvas API connectivity under current credentials,
- proves deterministic extraction of LMS structure,
- creates retained run-log artifacts for regression.

## What To Learn

- Safe read-first reproduction pattern before write operations.
- Consistent API call packaging and traceability in `run-log.yaml`.
- Repeatable snapshot output suitable for change detection and institutional review.
