# Save Memory

Immediately persist the current session context to memory.

## Process

Update `index.md` with current review context: active run, calibration summary, recurring issue patterns. Checkpoint `patterns.md` with any new specialist quality data, calibration adjustments, or effective feedback learnings. Checkpoint `chronology.md` with review cycle outcomes and human review results.

## Calibration Save

When the human reviewer provides feedback on Quinn-R's findings (accepts, rejects, or adjusts severity), immediately update:
- `index.md` calibration summary with the adjustment
- `patterns.md` with the specific calibration event and reasoning
- `chronology.md` with the timestamp and context

## Output

Confirm save with brief summary: "Memory saved. {brief-summary-of-what-was-updated}"
